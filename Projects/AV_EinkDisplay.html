The Build

The main star of the project is the Waveshare 7.5in monochrome e-ink display. With the power of the Rasperry Pi Zero W the display comes to life with data from CAIC's avalanche foracst and Winter Park Resort's live weather on the mountain. A 3D printed case brings all the components together into one compact package.

How's it work?

Every hour, the Rasperry Pi Zero W runs a python script to fetch the live information from the website. Once retrived, using the PIL python library the Pi Zero will render an image to display and refresh the e-ink display with the new data.
